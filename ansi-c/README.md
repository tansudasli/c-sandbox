# ansi-c

core C concepts 

## How to Run

`git clone ...`,

- `brew install gcc` or `xcode-select --install` for compiler
- `brew install cmake` to compile w/ cmake
- then,
    - `cd ansi-c`,
    - `./compile.sh`, and 
    - First `cd out/`,
      then Run as `./getInput` or other variations

Or, if you want, you can also run w/ `gcc src/xxx.c -o xxx && ./xxx`.
